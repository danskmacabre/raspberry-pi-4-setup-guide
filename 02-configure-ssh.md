# Configure SSH

## How to connect Raspberry Pi 4 Model B from remote computer using SSH

#### Prerequisite(s):
- [Enable SSH on your Raspberry Pi 4 Model B](./01-getting-started.md)
---
#### From Linux & Mac OS
---
```console
$ ssh pi@192.168.1.111
pi@192.168.1.111's password:
```
---
#### From Windows
---
```ternimal
C:\> ssh pi@192.168.1.111
pi@192.168.1.111's password:
```

> _Note: `ssh <USER-ID>@<IP-ADDRESS>`_
