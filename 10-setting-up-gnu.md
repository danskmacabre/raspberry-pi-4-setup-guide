# Setting Up GNU

## How to do a Raspberry Pi 4 Model B GNU setup

#### Install GNU
---
```console
pi@raspberry:~ $ sudo apt install raspberrypi-ui-mods
```

---
#### Uninstall GNU
---

```console
pi@raspberry:~ $ sudo apt-get remove --auto-remove --purge 'libx11-.*'
pi@raspberry:~ $ sudo apt-get autoremove --purge
```
